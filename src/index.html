Sec. /01

Constância leva a disciplina

O projeto surgiu de uma ideia de progressão. A vontade de documentar a jornada e, ao "final" dela, olhar para trás e ver quanto tempo, adversidades, passaram, foram maiores. Por isso a idealização do conceito.

De início, pareceu um pouco de redundância ao documentar a jornada de crescimento profissional (e de certo modo, pessoal), já que o próprio Github faz isso.

Mas isso ainda é e sempre será uma forma de aprendizado. A vontade de deixar documentado para um "eu do futuro", mesmo que pareça infantil, ainda é algo a se pensar.
